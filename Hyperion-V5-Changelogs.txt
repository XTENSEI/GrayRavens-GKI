## üöÄ Overview
**Hyperion-V5 Ultra** is a performance-oriented GKI kernel designed for enthusiasts who demand the absolute best from their hardware. By merging the latest upstream GKI changes with custom-tuned schedulers and networking algorithms, Hyperion delivers a "zero-lag" experience for both daily drivers and competitive gamers.

### üíé Key Highlights
* **Ultra-Low Latency:** 1000Hz polling for near-instant touch response.
* **Gaming Optimized:** Advanced congestion control and jitter reduction.
* **Stealth Root:** Integrated KernelSU-Next for clean, systemless management.

## ‚ö° Features & Optimizations

### üéÆ Performance & Gaming
* **500HZ Polling Rate:** Dramatically reduces input lag for competitive gaming.
* **Zen-Interactive Governor:** Intelligently predicts UI touch events to ramp up CPU clock speeds instantly, eliminating micro-stutters.
* **Full Preemption (PREEMPT):** Prioritizes real-time user tasks over background system processes.

### üß† Intelligence & Memory
* **Optimized Swappiness (10):** Maximizes use of physical RAM before utilizing swap/zRAM.
* **ART/Dalvik Boost:** App optimization utilizes 8 threads for significantly faster installation and boot times.
* **VM Tuning:** Adjusted VFS cache and dirty ratios to ensure large file transfers don't slow down the system.

### üåê Connectivity
* **BBRv2:** Google's latest congestion control for stable pings on unstable networks.
* **WireGuard:** Native kernel-level encryption for high-speed VPN performance with minimal battery drain.
* **TC-CAKE:** Advanced queue management to eliminate "bufferbloat" during heavy network usage.

## üì¶ Installation
> [!IMPORTANT]
> Always backup your `boot.img` or existing kernel before flashing.

1.  Download the latest **Hyperion-V5 Ultra** zip.
2.  Reboot to your custom recovery (TWRP/OrangeFox).
3.  Flash the `.zip` file.
4.  Reboot to System.
5.  *(Optional)* Install the [Kernel Manager](https://t.me/notsoAndriana/1881) to monitor stats.
